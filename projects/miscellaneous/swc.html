Scalable Web Cache

Implementation of scalable web cache using consistent hashing on top of redis. The source code is only compatible with windows as their is some windows specific code. To test, you need a windows machine. You also need to install redis (https://github.com/MSOpenTech/redis/releases).The redis servers store more than 10000 dummy key value pairs. The keys are distributed in three nodes initially. The nodes can be added or removed from the frontend view (views directory). You can also add key value pair to redis. The server on which key will be stored will be determined by consistent hashing. The communication between different clients takes place through REST web services.
