Fitbit Social Network

This is the application which I developed in my free time.The application uses sqlite for data storage, fragments, intent, pending intent and widgets. The notifications are managed through Intent Service. The backup of notes is done though XML. The notes are backed up on the native storage. It also uses Android speech to text api for writing notes. The UI has several activities. The main activity uses a view pager to show different notes.
